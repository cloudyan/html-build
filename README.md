# html-build

传统 html 站点打包工具，编译传统 html 静态资源到 cdn，并启用强缓存

## 解决方案

使传统 html 站点也能充分利用强缓存，cdn，并维护方便

1. 简单的解决方案，可以对将资源添加时间戳，并设置强缓存
2. 复杂但强大的解决方案，如下
   - 自动将资源添加 md5 命名后缀
   - 将静态资源上传到 cdn
   - 替换静态文件中引用的对应资源
   - 发布

## 使用

将站点放在 src 目录，并配置 cdn，默认替换 src 根目录下的资源引用

## 发布

1. 使用 gulp@3.9.1 依赖 node 版本需低于 10 版本，所以切换到 node 到 9.11.2 版本
   - `nvm use 9.11.2`
2. 安装依赖 `npm i`
3. `npm run deploy`
